STEP 2: Create Custom Fields in Salesforce
Salesforce comes with standard fields (Name, Email, Phone, etc.) but doesn't have VQMS-specific fields like Vendor_ID, Vendor_Tier, or SLA_Response_Hours. We need to create these as custom fields before importing data.

Naming Rule: Every custom field you create in Salesforce automatically gets __c appended to its API name. So if you name a field "Vendor ID", its API name becomes Vendor_ID__c.

2.1 How to Navigate to Custom Field Creation
This is the same navigation for ALL three objects. Repeat for each:
Step-by-step click path:

   -> This opens the Setup page in a new tab
