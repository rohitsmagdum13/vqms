# GDPR Compliance Checklist

- [ ] Data processing agreements with all third-party providers
- [ ] PII redaction before LLM processing (AWS Comprehend)
- [ ] Right to erasure implemented across all stores
- [ ] Data minimization in logging (no PII in logs)
- [ ] Encryption at rest (AES-256) and in transit (TLS 1.2+)
- [ ] Access control and audit trails for all data access
- [ ] Data retention policies enforced with TTLs
- [ ] Consent management for email processing
- [ ] Data breach notification procedures documented
- [ ] Privacy impact assessment completed
